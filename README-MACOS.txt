MARKER TIMESTAMPS - MACOS INSTALL

Use this file:

Marker-Timestamps-Premiere-Extension-macOS.zip

Steps:

1. Close Adobe Premiere Pro.
2. Extract Marker-Timestamps-Premiere-Extension-macOS.zip.
3. Open the extracted premiere-marker-timestamps folder.
4. Right-click or Control-click:

   Install on macOS.command

5. Choose:

   Open

6. Restart Premiere Pro.
7. Open:

   Window > Extensions > Marker Timestamps

If macOS says Apple could not verify the installer, read:

macOS - If Apple Could Not Verify.txt

If there is another problem, read:

SOLUTION - Install and Use Problems.txt
