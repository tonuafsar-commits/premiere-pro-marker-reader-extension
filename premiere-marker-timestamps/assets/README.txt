Optional success sound

If you own the rights to a sound file and want the extension to play it after Save TXT succeeds, place it in this folder as:

save-success.mp3

If this file is missing, the extension plays a short built-in chime instead.

This sound is loaded only by the extension panel UI after a TXT file is saved.
It is not imported into the Premiere Pro project, timeline, sequence, render
queue, or exported video/audio.
