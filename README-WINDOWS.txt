MARKER TIMESTAMPS - WINDOWS INSTALL

Use this file:

Marker-Timestamps-Premiere-Installer-Windows.exe

Steps:

1. Close Adobe Premiere Pro.
2. Double-click Marker-Timestamps-Premiere-Installer-Windows.exe.
3. If Windows shows Unknown Publisher, click:

   More info > Run anyway

4. Restart Premiere Pro.
5. Open:

   Window > Extensions > Marker Timestamps

If there is a problem, read:
